# flask_file_server
Web file server using Flask + JInja2

# Feature
* Login to user mode and admin mode
* File Browser
  * File Download
  * File Upload
  * File Delete
  * Directory Delete (Empty Directory Only)
  * Sort Directory List by name, size, date
* File Viewer
* User Management
